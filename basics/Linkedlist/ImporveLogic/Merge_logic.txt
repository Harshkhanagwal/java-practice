MERGE TWO SORTED LINKED LISTS

Concept:
Merge two already sorted linked lists into one sorted linked list by reusing existing nodes (do not create new nodes).

Pointers Used:

* dummy -> Temporary starting node of the merged list.
* tail  -> Always points to the last node of the merged list.
* list1 -> Current node of the first list.
* list2 -> Current node of the second list.

Algorithm:

1. Create a dummy node.
2. Set tail = dummy.
3. Compare list1.val and list2.val.
4. Attach the smaller node after tail.
5. Move the pointer (list1 or list2) from which the node was taken.
6. Move tail one step forward.
7. Repeat until one list becomes null.
8. Attach the remaining nodes.
9. Return dummy.next.

Pseudo Steps:

dummy = new Node()
tail = dummy

while(list1 != null && list2 != null)

```
if(list1.val < list2.val)
    tail.next = list1
    list1 = list1.next
else
    tail.next = list2
    list2 = list2.next

tail = tail.next
```

if(list1 != null)
tail.next = list1
else
tail.next = list2

return dummy.next

---

Dry Run

List1:
1 -> 3 -> 5

List2:
2 -> 4 -> 6

Step 1:
1 < 2
Merged:
1

Step 2:
3 > 2
Merged:
1 -> 2

Step 3:
3 < 4
Merged:
1 -> 2 -> 3

Step 4:
5 > 4
Merged:
1 -> 2 -> 3 -> 4

Step 5:
5 < 6
Merged:
1 -> 2 -> 3 -> 4 -> 5

Attach remaining:
6

Final:
1 -> 2 -> 3 -> 4 -> 5 -> 6

---

Time Complexity : O(m + n)

Space Complexity: O(1)
(Only one dummy node is used.)

---

Important Rules

1. Do NOT create new nodes.
2. Reuse the existing nodes.
3. Dummy node avoids handling the first insertion separately.
4. Tail always points to the last node of the merged list.
5. After one list ends, attach the remaining list in one step.
