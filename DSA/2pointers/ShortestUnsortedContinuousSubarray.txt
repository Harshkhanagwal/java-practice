
Given an integer array nums, you need to find one continuous subarray such that if you only sort this subarray in non-decreasing order, then the whole array will be sorted in non-decreasing order.

Return the shortest such subarray and output its length.

 

Example 1:

Input: nums = [2,6,4,8,10,9,15]
Output: 5
Explanation: You need to sort [6, 4, 8, 10, 9] in ascending order to make the whole array sorted in ascending order.
Example 2:

Input: nums = [1,2,3,4]
Output: 0
Example 3:

Input: nums = [1]
Output: 0



Think of this solution as finding:

How far disorder spreads

using two scans.

NOT traditional two pointers like:

left++
right--

But boundary detection.

Main Idea

A sorted array should satisfy:

left → right : always increasing
right → left : always decreasing

If this rule breaks:

element is misplaced
update boundary
Pass 1 → Find Right Boundary
int mxSeen = Integer.MIN_VALUE;

We move:

LEFT → RIGHT
Pointer Meaning

i is current pointer.

mxSeen means:

largest element seen so far
Logic
if(nums[i] >= mxSeen)

means:

current element is valid

Update max.

Else:

nums[i] < mxSeen

means:

this element should have appeared earlier

So array is unsorted till here.

Update:

r = i;
Example
[2,6,4,8,10,9,15]
Step-by-step
i = 0
2 >= -∞
mxSeen = 2
i = 1
6 >= 2
mxSeen = 6
i = 2
4 < 6

Problem found.

r = 2

Because 4 should be before 6.

i = 5
9 < 10

Again unsorted.

r = 5

Final:

right boundary = 5
Pass 2 → Find Left Boundary

Now move:

RIGHT → LEFT

Keep:

mnSeen

Meaning:

smallest element seen so far from right
Logic

If:

nums[i] <= mnSeen

valid.

Else:

nums[i] > mnSeen

means:

this number is too large

and should move right.

So:

l = i;
Example

Traverse backward.

i = 6
15 <= +∞
mnSeen = 15
i = 5
9 <= 15
mnSeen = 9
i = 4
10 > 9

Unsorted.

l = 4
i = 1
6 > 4

Again unsorted.

l = 1

Final:

left boundary = 1
Final Window
l = 1
r = 5

Subarray:

[6,4,8,10,9]

Sorting only this makes entire array sorted.



class Solution {
    public int findUnsortedSubarray(int[] nums) {
        

        int n = nums.length;

        int l = -1;
        int r = -1;

        int mxSeen = Integer.MIN_VALUE;

        for (int i = 0; i<n;i++){

            if(nums[i] >= mxSeen) {
                mxSeen = nums[i];
            }else{
                r = i;
            }
        } 


        int mnSeen = Integer.MAX_VALUE;

        for(int i = n - 1; i > -1;i--) {

            if(nums[i] <= mnSeen) {
                mnSeen = nums[i];
            }else{
                l = i;
            }
        }


        if(r  == -1 ) {
            return 0;
        }

        
        return r - l + 1;
    }
}
